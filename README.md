# Installation

## Cocoapods

Install Cocoapods if need be.

```bash
$ gem install cocoapods
```

Add `FaceTrigger` in your `Podfile`.

```ruby
use_frameworks!

pod 'FaceTrigger'
```

Then, run the following command.

```bash
$ pod install
```

## Manual

Copy `FaceTrigger` folder to your project. That's it.
